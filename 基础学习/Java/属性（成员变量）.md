属性vs局部变量

- 在类中声明的位置不同

​		属性（成员变量）：在类中声明的变量

​		局部变量：声明在方法内、方法形参、代码块内、构造器形参、构造器内部变量

- 权限修饰符

  属性：可以在声明属性时，使用权限修饰符，指定其权限

  局部变量：不可以使用权限修饰符

- 默认初始化值

  属性：有默认初始化值

  | 类型                      | 默认初始化值 |
  | ------------------------- | ------------ |
  | 整型(byte/short/int/long) | 0            |
  | 浮点型(float/double)      | 0.0          |
  | 字符型(char)              | 0('\u0000')  |
  | 布尔型(boolean)           | false        |
  | 引用型(类/数组/接口)      | null         |

​		局部变量：没有默认初始化值（意味着在声明/调用变量的同时要进行赋值）

- 在内存中加载的位置不同

  属性：加载到堆空间（非static）

  局部变量：加载到栈空间
